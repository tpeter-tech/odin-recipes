# odin-recipes

This is my very first web development project from The Odin Project foundations course. It is a simple recipes website built entirely using HTML.

## Skills Demonstrated
By completing this project, I have practiced and demonstrated the following skills:
* Using the terminal to navigate directories (cd, mkdir, touch).
* Initializing, committing, and pushing code changes with Git and GitHub.
* Writing clean and structured structural HTML code.
* Working with HTML text elements, paragraphs, headings, lists, and images.
